# Sonatype Nexus (Community Edition)

This application provides a Sonatype Nexus Community Edition instance configured to manage software artifacts.

## Architecture

- **Artifact Registry**: Powered by a Google Firebase Project with 5GB of storage.
- **Hosting**: Deployed and hosted on Hugging Face Spaces.

## Purpose

The purpose of this app is to provide a lightweight, cloud-hosted artifact repository for the Nucleus project, leveraging free-tier cloud resources for storage and compute.
